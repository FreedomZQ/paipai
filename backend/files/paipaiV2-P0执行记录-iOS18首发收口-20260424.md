## 1. 本次决策

为尽快上线 Apple Store，首发版本改为 **iOS/iPadOS 18.0+**。

目的：

- 避免继续承诺 iOS 17.x 上未完成验证的设备端翻译能力。
- 保留现有页面展示风格，不做大范围 UI 重设计。
- 降低个人开发者首个 App 的适配成本、运维成本和审核解释成本。
- 将首发卖点收口到更稳的能力：本地 OCR、本地朗读、句卡学习、复习、家长区、Apple 登录、IAP、账号删除、可选云 OCR/TTS、云同步。

## 4. 低运维 / 低法律风险 / 转化与留存建议

### 低运维

- 首发少做服务端强依赖：设备 OCR 和设备朗读保持默认可用。
- 云 OCR/TTS 只做用户主动选择和配额控制，不作为基础使用必需项。
- release gate 继续 fail-closed：缺配置不放行。

### 低法律风险

- 不再承诺 iOS 17.x 设备端翻译。
- 不再在文档里宣称未验收完成的“30 天自动删除已完成”。
- 隐私材料必须真实描述 Apple 登录、IAP、云 OCR/TTS、PowerSync、使用时长。

### 高付费转化

- 免费版保留可体验的本地 OCR/TTS + 少量句卡。
- 付费点放在多孩子、更多句卡、历史统计、云同步、云 OCR/TTS 额度。
- Paywall 必须使用 StoreKit 本地化价格，减少审核和用户质疑。

### 留存率

- 保留每日任务、复习、连续学习、家长周报。
- 首发先保证稳定闭环，不急着扩太多语言。
