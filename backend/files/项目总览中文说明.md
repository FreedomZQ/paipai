# AppHub Backend 项目中文说明文档

生成时间：2026-04-28 09:35:22

## 1. 项目定位
该项目是基于 Spring Boot 3 的统一多 App 后端，核心目标是把认证、计费、配置中心、AppStore 回调、PowerSync 同步、发布门禁等公共能力沉淀为系统模块，同时为 reading、saving、fitmystery 等具体 App 提供独立业务接口与模块化扩展点。

## 2. 技术栈
- Java 17 / Spring Boot Web：提供 REST API。
- MyBatis-Plus：负责数据库实体和 Mapper 访问。
- PostgreSQL：主业务数据库。
- Flyway：管理数据库 migration。
- Redis：预留缓存或会话相关基础设施。
- springdoc-openapi：提供 OpenAPI/Swagger 文档能力。
- Nimbus JOSE JWT：用于 Apple 登录、JWT、JWS 校验相关能力。

## 3. 目录结构说明
- 
`src/main/java/com/apphub/backend/common`
：通用配置、响应体、过滤器、工具类。
- 
`src/main/java/com/apphub/backend/shared`
：跨系统模块共享能力，目前主要是 Apple JWT/JWK。
- 
`src/main/java/com/apphub/backend/sys`
：统一后端系统能力，包括 app 编排、auth、billing、appstore、configcenter、powersync。
- 
`src/main/java/com/apphub/backend/apps/common`
：多 App 共享抽象，如 AppModule、AppModuleRegistry、AppPowerSyncAdapter、版本策略。
- 
`src/main/java/com/apphub/backend/apps/reading`
：阅读陪伴 App 专属模块，重点包含 PowerSync、OCR/TTS 云能力。
- 
`src/main/java/com/apphub/backend/apps/saving`
：Saving/存钱类 App 专属模块，包含账户、配置、财务记录、报表、兼容计费接口。
- 
`src/main/java/com/apphub/backend/apps/fitmystery`
：FitMystery App 专属模块，包含账户、活动、盲盒、购买、报告等。
- 
`src/main/resources/apps/*/app-definition.yml`
：每个 App 的产品定义、AppStore/Apple 配置、版本策略和发布配置。
- 
`src/main/resources/db/migration`
：数据库版本脚本。
- 
`powersync`
：PowerSync 同步规则与部署配置。
